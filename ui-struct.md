# ui-struct.md

## Композиция

- `model-stage` — полноэкранная сцена.
- `model-view` — 3D-область (главная часть экрана).
- `app-header` — две кнопки вверху:
- `book` (список тем),
- `theme` (dark/light).
- `topic-navigation` — всплывающий список тем.
- `lesson-panel` — отдельная карточка выбранной темы.

## Состояние приложения

- `theme`: `dark`/`light`.
- `isBookOpen`: открыт ли список тем.
- `activeTopicId`: активная тема.
- `isCardOpen`: открыта ли карточка темы.
- `isLandscape`: текущая ориентация экрана.

## Поведение

- Кнопка `book`: открывает/закрывает список тем.
- Выбор темы:
- открывает/обновляет единственную карточку,
- повторный клик по той же теме закрывает карточку.
- `X` в списке закрывает список.
- `X` в карточке закрывает карточку.
- Модель всегда остаётся основной и обновляется под активную тему.

## Визуальная система

- Цвета: white/black/red; gray только в модели.
- Текст без тяжёлых табличных контейнеров.
- Типографика: SF Pro Text (или системный Apple-like fallback).
