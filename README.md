# О проекте

`easy-g` - офлайн-доступный PWA-учебник по геометрии для 5-11 классов.
Темы курса хранятся в JSON-карточках в каталоге `cards/`, а каждая карточка
может быть показана как текстом, так и интерактивной 3D-сценой.

На текущем этапе в приложении сохранена grid-оболочка исходного проекта:
верхняя панель, боковой блок и нижняя диагностика пока остаются как
временный каркас. Основная рабочая часть уже переведена на карточку easy-g
и отрисовывается собственным локальным 3D-движком без внешних зависимостей.

## Основные блоки интерфейса

- Область 3D-карточки с текстом и интерактивной сценой
- Панель режимов исходного шаблона
- Боковая панель исходного шаблона
- Нижняя диагностическая панель исходного шаблона

## Технологический стек

- JavaScript (ECMAScript 2022+)
- Нативный DOM API
- Canvas 2D с собственной 3D-проекцией
- CSS Grid/Flex
- JSON-карточки тем

## Принципы реализации

- Каждая тема хранится в отдельном JSON-файле в `cards/`.
- 3D-движок строит сцену из примитивов карточки и не зависит от текстур.
- Модель всегда центрируется относительно сцены, а масштаб меняет только
  дистанцию до камеры, не смещая точку фокуса.
- Визуализация карточек использует ограниченную палитру: черный и красный.
- Крупные UI-блоки реализованы отдельными компонентами-классами.
- Структура репозитория поддерживается в `struct.md` и `ui-struct.md`.
