# easy-g

`easy-g` — офлайн PWA по геометрии в формате карточек для запоминания определений, свойств и теорем, с визуальным образом через 3D-модель.

## Ключевая идея

- Главная область экрана — 3D-модель.
- Текст и управление накладываются поверх, без тяжёлых панелей.
- Список тем открывается через кнопку книги.
- Карточка темы открывается отдельно и показывает:
- определение;
- свойства;
- теорему.

## Визуальные правила

- Основа палитры: `black`, `white`, `red`.
- В модели дополнительно используется `gray` для вторичных элементов.
- Поддерживаются две темы: `dark` (по умолчанию) и `light`.
- Типографика ориентирована на `SF Pro Text`.

## Шрифты

Для локального подключения добавьте в `fonts/`:

- `SFProText-Regular.woff2` или `SFProText-Regular.woff`
- `SFProText-Semibold.woff2` или `SFProText-Semibold.woff`
- `SFProText-Bold.woff2` или `SFProText-Bold.woff`

Если файлов нет, используется системный стек шрифтов Apple/санс-сериф fallback.
